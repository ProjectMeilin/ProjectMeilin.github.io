## Update Pages
source ENV/bin/activate
git checkout source         ( then editing your articles in content/*.md )

make html
make serve                  ( check at localhost:8000 )

git branch -D gh-pages
git branch gh-pages
ghp-import output
git checkout master
git merge gh-pages
git push --all
